# Lunarglow

Lunarglow is an open source CLI client for [Sunshine](https://github.com/LizardByte/Sunshine) and NVIDIA GameStream on Linux PCs, including x86 and ARM systems. Lunarglow allows you to stream your full collection of games and applications from your host PC to another PC.

## Documentation

More information about the upstream protocol and shared code is available from the [Moonlight project](https://moonlight-stream.org/).
